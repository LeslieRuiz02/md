# Proyecto: El Micrófono Mágico de los Backyardigans

Este repositorio contiene materiales didácticos y un juego interactivo en HTML diseñado para abordar el **Campo Formativo de Lenguajes** de la Nueva Escuela Mexicana (NEM), específicamente para preescolar.

## Contenido

1. **`juego_turnos_backyardigans.html`**: Un juego interactivo contenido en un único archivo. No requiere conexión a internet para funcionar una vez descargado y contiene todos los elementos visuales y auditivos dentro de sí mismo. Ideal para enviarse por WhatsApp.
2. **`planeacion_didactica.md`**: Planeación docente formal que alinea la actividad con la NEM (Fase 2, Lenguajes, PDA: *Escucha con atención a sus pares y espera su turno para hablar*).
3. **`instrucciones_para_padres.md`**: Guía amigable para que las familias utilicen el juego en casa.
4. **`todo.md`**: Lista de mejoras y tareas pendientes para el proyecto.

## Instrucciones Técnicas

Para usar el juego:
1. Abre el archivo `juego_turnos_backyardigans.html` en cualquier navegador web (Chrome, Safari, Firefox, Edge).
2. Si lo envías por WhatsApp, los padres solo tendrán que tocar el archivo y su teléfono lo abrirá automáticamente en el navegador por defecto.
3. El juego generará sonidos usando el dispositivo, asegúrate de que el volumen esté encendido.

## Propósito Pedagógico
Fomentar la escucha activa y la inhibición de impulsos (no interrumpir) mediante el juego de turnos mediado por personajes conocidos por los niños.
